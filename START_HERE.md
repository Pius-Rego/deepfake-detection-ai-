## What changed

This repository has been repositioned from a generic deepfake-detection project into **DRISHTI**, a defence-oriented prototype for:

- real-time military deepfake detection
- India-specific impersonation checks
- disinformation weaponisation scoring
- analyst-ready reporting

## Best demo path

1. Start the Django app:

```bash
cd "Django Application"
python manage.py runserver
```

2. Open the console:

```text
http://127.0.0.1:8000/
```

3. Upload a sample clip.

4. Walk the judges through:

- threat verdict
- fused signals
- impersonation matches
- weaponisation score
- generated analyst report
